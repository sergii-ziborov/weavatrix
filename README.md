# Weavatrix

**Code graph & blast-radius MCP server for AI coding agents.**

Weavatrix builds a dependency graph over any local repository (tree-sitter,
no native compilation) and serves it to Claude Code / Codex over the Model
Context Protocol: change impact, transitive dependents, health audit, clone
detection, coverage mapping, endpoint inventory.

Supported languages: JavaScript, TypeScript/TSX, Python, Go, Java, HTML, CSS.

> **Status: pre-release.** Packaging in progress — not yet published to npm.
